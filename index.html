Latihan Web Design

1. Perbedaan Web, Web Browser, Web Server, dan Text Editor

Web adalah kumpulan halaman yang saling terhubung dan dapat diakses melalui internet menggunakan alamat URL.

Web Browser adalah aplikasi yang digunakan untuk membuka dan menampilkan halaman web, contohnya Google Chrome, Mozilla Firefox, dan Microsoft Edge.

Web Server adalah sistem atau komputer yang menyimpan file website dan mengirimkannya ke browser saat diminta oleh pengguna melalui internet.

Text Editor adalah aplikasi yang digunakan untuk menulis dan mengedit kode program seperti HTML, CSS, dan JavaScript, contohnya Visual Studio Code dan Notepad++.

2. Aplikasi yang Digunakan untuk Membangun Website

Beberapa aplikasi yang dapat digunakan dalam pembuatan website antara lain:

  • Visual Studio Code → text editor modern dengan banyak fitur
  • Sublime Text → ringan dan cepat digunakan
  • Notepad++ → sederhana dan mudah dipakai
  • Adobe Dreamweaver → tools desain dan coding website
  • XAMPP → untuk menjalankan server lokal

3. Tata Cara Menyimpan Halaman Web

Langkah-langkah menyimpan file HTML:

  1. Klik menu File
  2. Pilih Save As
  3. Tentukan lokasi penyimpanan
  4. Beri nama file dengan ekstensi .html (contoh: latihan1.html)
  5. Pilih tipe file All Files (jika ada)
  6. Klik Save

4. Pendapat tentang Perkembangan Website ke Depan

Menurut saya, perkembangan website ke depan akan semakin maju dan canggih. Website akan menjadi lebih interaktif dengan adanya animasi dan teknologi modern seperti AI (Artificial Intelligence). Selain itu, website akan lebih responsif sehingga bisa digunakan dengan baik di berbagai perangkat seperti smartphone, tablet, dan komputer. Kecepatan akses juga akan semakin meningkat dan keamanan website akan lebih diperhatikan.

Semua jawaban di atas telah dibuat menggunakan bahasa HTML dan ditampilkan melalui web browser. Tampilan halaman ini juga menggunakan CSS yang dipisahkan dari file HTML untuk mengatur desain.
